チャットボタンの比較テスト

上部のタブで案A / 案Bを切り替えできます。
右下に表示されるボタンをタップするとチャットが開きます。

※PC幅（769px以上）ではどちらも横長バナーが表示されます。
スマホ実機またはブラウザの幅を768px以下に狭めてご確認ください。

案A: 丸型ボタン

形状: 直径60pxの丸
アイコン: 吹き出し（白）
立体感: グラデーション + 上辺ハイライト + ピンク影
アニメーション: 光の筋が横切るシマー効果
メリット: コンパクト。フッターに被りにくい
デメリット: テキストがないので何のボタンか伝わりにくい

案B: ピル型バナー

形状: 角丸ピル型（テキスト幅に合わせて可変）
アイコン: 吹き出し（白）+ 「AIに質問」テキスト
立体感: グラデーション + 上辺ハイライト + ピンク影
アニメーション: 光の筋が横切るシマー効果
メリット: テキスト付きなのでタップ率が高い
デメリット: 丸型より面積が大きい

共通仕様

位置: right / bottom はサイトに合わせて調整可能
PC表示: 横長バナー画像（現行と同じ）
スマホ表示: 上記いずれかのボタン
チャット開閉: タップでDifyチャットが全画面で開く
